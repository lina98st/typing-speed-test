# Typing Speed Test

A typing speed test application built with React and Vite.

**Challenge by [Frontend Mentor](https://www.frontendmentor.io/challenges/typing-speed-test)**

## Features
- Multiple difficulty levels (Easy, Medium, Hard)
- Two test modes: Timed (60s) and Passage mode
- Real-time WPM and accuracy tracking
- Personal best tracking with localStorage
- Visual feedback for typing errors

## Tech Stack
- React
- Vite
- CSS
